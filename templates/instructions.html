Formatting Your .txt File

For the best results, please format your book text file according to these simple rules:

  • Separate each paragraph with a single blank line (press Enter twice).
  • To define a chapter, place its title on its own line in the format {-Chapter Title-}.
  • To include an image, place its URL on its own line in the format [IMAGE: https://path.to/your/image.jpg]. Ensure the URL is publicly accessible.

Example:

{-Chapter 1: The Adventure Begins-}

This is the first paragraph of the first chapter. It describes the setting and introduces the main character.

This is the second paragraph. It continues the narrative.

[IMAGE: https://i.imgur.com/example.jpeg]

{-Chapter 2: A New Challenge-}

This is the first paragraph of the second chapter.
